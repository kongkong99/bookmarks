## 项目简介

通过地图可视化操作，进行 WGS84、GCJ02、BD09 坐标的在线转化，这三种坐标几乎可以满足 90%以上的需求。

## 项目链接

- [仓库地址](https://github.com/kongkong99/coordinate)
- [在线链接](https://github.com/kongkong99/coordinate)

## IFrame 在线预览

```jsx | inline
import React from 'react';
import Iframe from './compoents/IFrame';

export default () => (
  <Iframe
    src="https://kongkong99.github.io/coordinate/#/"
    minWidth={1200}
  ></Iframe>
);
```
